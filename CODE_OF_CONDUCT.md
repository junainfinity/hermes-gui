# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind. Assume good intent.
- Technical disagreement is welcome; ad hominem is not.
- Harassment, discrimination, and personal attacks are grounds for removal.
- Report concerns by opening a private GitHub issue addressed to the
  maintainers, or emailing the address in `MAINTAINERS.md`.

## Enforcement

Maintainers are responsible for clarifying and enforcing these standards. They
may remove comments, commits, issues, and contributors that violate the code.

Full text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
